## Preparation
Download [git](https://git-scm.com/downloads) and [Node.js](https://nodejs.org/en/)

## Installation & Usage
> Open CLI ($ Command Line Interface)

    $ git clone https://github.com/DahisC/AI_bot.git
    $ cd ai_bot
    $ npm install

    $ node src/index
    (Browser) http://localhost:9001
